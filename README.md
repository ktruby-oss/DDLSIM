# DDLSim-Lab Project Website

This repository hosts the official project website and documentation for **DDLSim-Lab**, an open-source research project designed to simulate, analyze, and optimize distributed deep learning systems.

## 🌐 Live Website
**[Visit the DDLSim-Lab Website](https://ktruby-oss.github.io/DDLSIM/)**

## 🔬 Main Research Repository
The core simulation engine, fault injection modules, and AI-driven control plane code can be found in the main repository:
**[ktruby-oss/DDLSim-Lab](https://github.com/ktruby-oss/DDLSim-Lab)**

## 💻 Website Architecture
The site is built with modern web technologies focusing on performance, accessibility, and a clean user experience (UI/UX) suitable for academic research projects:
- **HTML5 & CSS3**: Custom styles with smooth CSS animations and transitions.
- **Bulma CSS Framework**: For responsive grid layouts and utility classes.
- **FontAwesome**: For scalable vector icons.
- **Unsplash Imagery**: High-quality, relevant photography to illustrate complex technical concepts.

### Features
- Sidebar Navigation for seamless browsing.
- Dedicated pages for Home, About & Call for Partners, Research & Experiments, and Documentation.
- Embedded BibTeX citation block.
- Mobile-responsive design.
